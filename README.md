# 📘 Git/GitHub Cheat Sheet

Git and GitHub are among the most widely used tools for **version control** and **collaborative software development**.

Git maintains the complete history of your project by tracking every change made to the source code. It records **what changed, who made the change, and when it was made**, making it easy to review project history, collaborate with team members, and restore previous versions whenever needed.

Whether you're working individually or in a team, Git helps you manage your code efficiently, while GitHub provides a platform to host repositories, collaborate through pull requests, track issues, and manage projects.

---

## 📚 About This Repository

This repository is designed as a **quick reference guide** for Git and GitHub commands.

It covers commonly used commands along with their purpose, making it easier for:
- 🎓 Students learning Git for the first time
- 👨‍💻 Developers preparing for interviews
- 🚀 Contributors working on open-source projects
- 💼 Professionals who need a quick command reference

The cheat sheet is organized in a simple and beginner-friendly format so you can quickly find the command you need.
