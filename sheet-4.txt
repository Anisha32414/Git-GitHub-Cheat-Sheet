## Merge commands:-

* git diff "branch_name"                      // used to compare the difference like new files, updation, deletion etc.
* git merge "branch_name"                     // used to merge two brances


// We can use PR(pull request) as well on github to merge two brances.

* git pull origin main                       // used to copy exact code from remote repo to local repo.
